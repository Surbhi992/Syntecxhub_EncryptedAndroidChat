# 🔐 Encrypted Android Chat App

A simple Android application that demonstrates secure message encryption and decryption using AES (Advanced Encryption Standard). This project is designed for learning basic cryptography concepts in Android development.

---

## 📱 Features

* 🔒 AES Encryption & Decryption
* 💬 Simple Chat Interface
* ⚡ Lightweight and easy to use
* 📦 No external dependencies required

---

## 🛠️ Technologies Used

* Java
* Android Studio
* XML (UI Design)
* AES Encryption Algorithm

---

## 🚀 How to Run the Project

1. Clone or download this repository
2. Open the project in Android Studio
3. Let Gradle sync complete
4. Connect an emulator or Android device
5. Click **Run ▶️**

---

## 📂 Project Structure

```
app/
 └── src/
     └── main/
         ├── java/com/example/encryptedchat/
         │     ├── MainActivity.java
         │     └── CryptoUtils.java
         ├── res/values/
         │     ├── strings.xml
         │     └── themes.xml
         └── AndroidManifest.xml
```

---

## 🔑 How It Works

* User enters a message
* Message is encrypted using AES
* Encrypted message is displayed
* Can be decrypted back to original text

---

## 📌 Note

This project is for educational purposes and demonstrates basic encryption techniques. It is not intended for production-level security.
